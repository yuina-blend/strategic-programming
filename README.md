# 課題2のやつ

p>q>r>sの順序が変化するので、全パターンにおいてどの手を出すのが最適か調査する。

先頭のアルファベットで場合分けして分担する。

## 結論

sが先頭の場合 -> 常に0

pが先頭の場合 -> 常に1

q, rが先頭の場合 -> s, pの大小関係で決定

q, r(s>p) -> 常に0

q, r(p>s) -> 常に1
